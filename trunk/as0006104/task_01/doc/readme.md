### How to build the project?

1. The first you need to clone this repository to your computer.

2. Go to the folder "trunk\as0006104\task_01\src".

3. Run the command line and type 6 commands :

```console 
mkdir build
cd build
cmake ..
cmake --build .
cd MainFile\Debug
.\Main.exe
```
